# widget — Chatbot Widget em React 💬

Componente de **widget de chatbot** reutilizável, desenvolvido em **React**. Encapsula a interface de um chat flutuante (container + balão) que pode ser integrado a uma aplicação web.

## ✨ Características

- Componentes `ChatBoot` e `ChatBotContainer` reutilizáveis
- Interface em modal flutuante (`react-modal`)
- Estilização com **styled-components** / **Emotion** / **MUI**
- Navegação com **React Router**

## 🚀 Como executar

```bash
git clone https://github.com/limongi1234/widget.git
cd widget
yarn install   # ou npm install

```
